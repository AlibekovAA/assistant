# Ассистент

Легковесный локальный голосовой помощник, написанный на Python.

Проект ориентирован на минимальное потребление ресурсов, высокую скорость работы и модульную архитектуру.

## Цели проекта

* Минимальное потребление ресурсов.
* Постоянная работа в фоновом режиме.
* Низкая нагрузка на процессор и оперативную память.
* Распознавание речи.
* Озвучивание ответов.
* Активация по ключевой фразе.
* Современная модульная архитектура.
* Использование Python 3.13.
* Кроссплатформенная архитектура.

## Структура проекта

```text
src/
└── assistant/
    ├── audio/
    ├── core/
    ├── stt/
    └── tts/
```

## Разработка

Синхронизация зависимостей:

```bash
uv sync
```

Форматирование кода:

```bash
make format
```

Проверка линтером:

```bash
make lint
```

Полная проверка проекта:

```bash
make check
```

Запуск приложения:

```bash
uv run assistant
```
